# Snake Game (Python + Tkinter)

This is a simple Snake game built in Python using `tkinter`.  
It was created as part of an assignment to practice Git concepts such as:

Basic features:
-Start Menu (Start Game / Exit)
-Background music (MP3) + sound effects
-Snake grows when eating food
-Increasing speed as difficulty rises
-Score display
-Game Over menu (Try Again / Exit)
-Pause / Resume functionality

## How to Run

1. Install Python
2. Install required libraries:

   pip install pygame
